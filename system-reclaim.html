System Reclaim

Daily control panel for the personal factory.

A gamified habit and project tracker styled as a worn industrial HMI. Real-world habits map to factory subsystems you monitor and advance every day. Habit tracker first, factory restoration second.

What it is

System Reclaim turns daily habits into systems work. Five subsystems — movement, recovery, fuel, assembly, and will — map to real-world routines: exercise, sleep, protein intake, project effort, and family time. Each one has a control and a status indicator on a single panel, and each day either advances them or lets them drift back toward fault.

The longer arc is a factory restoration game called Fallen Factories. The near-term build is the panel itself: a stylized productivity dashboard that earns its aesthetic by actually doing its job. Built carefully, not rushed.

Five subsystems

DRIVE

Exercise. A 7-day rolling counter rewards consistency over intensity, and missed days dim the indicator instead of penalizing it.

POWER

Sleep. Hours logged drive the panel's own state — visible in the bezel and the boot animation on the next session.

INTAKE

Protein. Daily grams against a target, tracked simply because the simplest input that holds is the one that gets logged.

ASSEMBLY

Project effort. A dropdown of active projects (CMMS, this website, NCCER Level 4, Master's coursework, AIMMMES). One project per day, marked done or not.

WILL

Family time. The most important subsystem dressed as the simplest control: present today, yes or no.

Tech & build

  • Next.js 16 with React 19 and TypeScript
  • Tailwind CSS v4 with custom industrial styling: CRT bezel, phosphor text, scanline overlay, procedural noise grain, corner rust tints, boot animation
  • Local-first state via localStorage — no backend, no account required
  • Designed mobile-first; the panel scales cleanly from phone to wide desktop
  • Versioned at v0.2 (April 2026): wear textures, bezel, pulse, caution stripe, boot animation
